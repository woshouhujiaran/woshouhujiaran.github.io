雅思写作 Task 1：图表客观描述写作框架

发表于2026-04-23|更新于2026-04-23|分享
|浏览量:

一、核心原则

IELTS Writing Task 1 的目标不是评论数据，而是客观复述图表信息。写作时只做三件事：识别图表、提炼规律、组织对比。全文应清晰、简洁、可追溯到原图，避免个人观点与图外推断。

二、先判断图表类型

常见类型包括 line graph、table、pie chart、bar chart、map、process diagram。

  1. 动态图（有时间轴）：先抓整体趋势（increase / decrease / fluctuate），再分组。
  2. 静态图（无时间轴）：先横向比较数据高低与差距，再分组。
  3. 混合图：分别描述每张图，再在结尾建立联系。
  4. 地图题：先主要变化，再次要变化，最后补充不变项。
  5. 流程图：按阶段写，突出顺序和细节。

三、按图表类型拆分结构

1. 动态图（line/bar/table with time）

推荐结构：Summary + Group1 + Group2 + Conclusion

  1. Summary：图表类型 + 对象 + 整体趋势（increase / decrease / fluctuate，可写相反趋势）。
  2. Group1：按共性分组（同向变化、幅度接近、同区间），写起点、终点、主要涨跌。
  3. Group2：写另一组并突出 turning points、intersections、highs and lows。
  4. Conclusion：客观补充整体关系或相反趋势，不加图外推断。

句式（优先）：

  1. Subject rose/increased/climbed from ... to ...
  2. There was a slight/steady/sharp decline in ...
  3. ... peaked at ... before falling to ...
  4. ... remained stable at ... and then edged up to ...

连接词（动态）：

over the period、from ... to ...、before doing、and then、followed by、after which、by the end of the period

高频词（动态）：

rise、increase、climb、grow、drop、fall、decline、fluctuate、dip、level off、peak、bottom out、remain stable

2. 静态图（no time）

推荐结构：Summary + Grouping Comparison + Cross-group Comparison + Conclusion

  1. Summary：列出核心对象并给出初步高低对比（谁最高、谁最低、差距大致如何）。
  2. Grouping Comparison：按数值区间或特征共性分组，先组内比较，再写组内差异。
  3. Cross-group Comparison：比较组间差距、占比差异或集中程度。
  4. Conclusion：补一句客观总体特征（如分布集中/分散），不表达主观观点。

句式（优先）：

  1. X accounted for the largest proportion, while Y had the smallest share.
  2. The figure for A was higher/lower than that for B.
  3. A and B were similar, at ... and ... respectively.
  4. Combined, X and Y made up ... of the total.

连接词（静态）：

in contrast、whereas、while、compared with、respectively、similarly、by comparison

高频词（静态）：

proportion、percentage、share、account for、constitute、make up、majority、minority、respectively

3. 混合图（static+dynamic / dynamic+dynamic / static+static）

推荐结构：Overall Summary + Chart A + Chart B + Linkage Conclusion

  1. Overall Summary：说明两张图的主题和各自趋势方向。
  2. Chart A：按该图类型的规则写关键特征。
  3. Chart B：同上，避免重复句式。
  4. Linkage Conclusion：指出两图趋势是否同步、相反或仅部分相关（只基于图内信息）。

句式（优先）：

  1. The line graph indicates..., while the pie chart shows...
  2. A similar pattern can be seen in..., whereas ... follows the opposite trend.
  3. X increased over time, and this was accompanied by a higher share of Y.
  4. Taken together, the two charts suggest that ...

连接词（混合）：

meanwhile、in parallel、similarly、by contrast、whereas、taken together

高频词（混合）：

pattern、trend、correspond、mirror、opposite trend、association、overall relationship

4. 地图题（map）

推荐结构：Overview + Major Changes + Minor Changes + Unchanged Features

  1. Overview：时间范围（若有）+ 总体变化方向（开发增强、设施扩张、功能转换等）。
  2. Major Changes：先写核心结构变化。室外可先以 roundabout/circular road 为参照，优先写用地与交通主干变化。
  3. Minor Changes：写次要建筑、路径、绿地等调整。
  4. Unchanged Features：补充保留区域或未改动设施，增强完整性。

句式（优先）：

  1. The most noticeable change is that ... was converted into ...
  2. The road was extended, followed by the construction of ...
  3. ... was demolished/removed, while ... remained unchanged.
  4. Compared with the earlier map, the area became more ...

连接词（地图）：

compared with、over the period、in the north/south/east/west、at the center、followed by、by contrast

高频词（地图）：

convert(ed) into、redevelop(ed)、expand(ed)、extend(ed)、construct(ed)、rebuild/redevelop、demolish(ed)、replace(d)、remain unchanged

5. 流程图（process diagram）

推荐结构：Summary + Phase 1 + Phase 2 (+ Phase 3) + End State

  1. Summary：写流程起点、终点与总阶段数。
  2. Phase 1：原料/输入与第一步处理。
  3. Phase 2：中间转化步骤，强调顺序与条件（before doing, and then, followed by）。
  4. End State：最终产物或输出去向，可补充循环关系（若图中存在）。

流程图通常控制在约 150 词，其余图表约 200 词。

句式（优先）：

  1. The process begins with ... and ends with ...
  2. In the first stage, ... is/are ...
  3. At the next stage, ... is then ...
  4. After which, ... is transferred to ...
  5. In the final stage, ... is produced/delivered.

连接词（流程）：

initially、in the first stage、subsequently、after which、at this point、thereafter、finally

高频词（流程）：

stage、step、procedure、process consists of、be processed、be transferred、be heated/cooled、be stored、be delivered、cycle repeats

四、时态与用词提醒

  1. 动态图：以过去时或现在时描述时间变化，避免同一动词反复出现。
  2. 静态图：以一般现在时为主，重点放在比较与占比。
  3. 地图题：常用过去对比现在的时态组合（如 was converted into）。
  4. 流程图：多用一般现在时和被动语态（is processed, is transferred）。

五、写作检查清单（通用）

  1. 是否完整覆盖关键特征，而非罗列所有数字。
  2. 是否先总后分，且分组逻辑清晰。
  3. 是否避免重复题目原词，可用同义替换。
  4. 是否保持客观，不出现主观评价。
  5. 字数是否约 200 词（流程图约 150 词）。

按以上“分图表结构”写作，可以在不加入主观观点的前提下，稳定输出结构清晰、信息完整的 Task 1 描述。
